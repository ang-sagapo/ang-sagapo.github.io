p=a+b;q=a−bp = a+b;q = a-b p=a+b;q=a−b

在成功分解的情况下，得出 a 后，令

b2=a2−nb2 = a^2 - n b2=a2−n

# 第五题：

第五题为一个 flag 分成两半来解

# flag1

此为 n 很大而 e 很小的 rsa 解密，对 n 进行分解采用低指数加密

# 低指数加密原理：

当选取素数为 e = 3 时，加密公匙变为：

m3≡c(modn)m^3 ≡ c(mod n) m3≡c(modn)

对此式进行变换可知

由 cmodn=MOD nm3modn=MOD由\ c mod n = MOD \ n\\m^3 mod n = MOD 由 cmodn=MOD nm3modn=MOD

m3=c+k∗nm=(c+k∗n)(1/3)m^3 =c + k*n\\m = (c+k*n)^(1/3) m3=c+k∗nm=(c+k∗n)(1/3)

所以可以从小到大对 k 进行枚举，直到开方开出整数为止

dp=d mod(p−1)dp = d\ mod(p-1) dp=d mod(p−1)
